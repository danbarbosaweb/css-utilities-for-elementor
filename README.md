# 🎨 CSS Utilities for Elementor

![GitHub repo size](https://img.shields.io/github/repo-size/danbarbosaweb/css-utilities-for-elementor)
![GitHub stars](https://img.shields.io/github/stars/danbarbosaweb/css-utilities-for-elementor?style=social)

✨ Uma coleção de **snippets CSS reutilizáveis** para uso direto no Elementor — focada em personalização avançada, animações, microinterações e efeitos visuais modernos.

Aqui você encontra códigos prontos para copiar e colar, sem complicação, sem plugins desnecessários.

---

## 📂 Estrutura do Repositório

Cada pasta contém utilitários organizados por categoria:

- 📏 **scrollbar** → Estilos personalizados de barra de rolagem  
- 🔘 **buttons** → Hover, transições e efeitos interativos  
- 🃏 **cards** → Elevação, sombras e animações de foco  
- 🌀 **mouse-cursor-effect** → Cursor personalizado com CSS + JS  
- 🗂 **assets** → GIFs e mídias de demonstração  

Cada utilitário pode conter:
- CSS
- JavaScript (quando necessário)
- README próprio com instruções

---

## ⚙️ Como usar

1️⃣ Abra o Elementor no seu site WordPress  
2️⃣ Vá em **Avançado → Custom CSS** ou use um Widget HTML  
3️⃣ Copie o snippet desejado  
4️⃣ Cole e ajuste conforme necessário  

Simples assim. Sem mistério.
